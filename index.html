晚上书架 (Lost city's bookself)
晚上书架简介
—— An application for books

这是一个为“书痴”而生的app。

在信息爆炸的年代，我们需要为我们自己筛选信息，构建我们的知识系统。这就是我们这款app的目标。而我们的产品需要回答以下问题

1、如何去管理我们的电子书资源。实际上，我的电子书文件，比如pdf，到处都是。我的电脑里有，手机里面也有，kindle也有。各种格式，非常零乱。非但如此，散落在不同的地方，有的时候，我要寻找某一本书，手机也有文件，电脑也有重复的文件。 我们提供了这种解决方案： 我们把需要的文件，归档到电脑的某个文件目录下。然后设置本应用的电子书目录的路径，双击打开本应用。启动应用后，在浏览器输入相应的ip地址就可以阅读了，实现局域网内的所有设备，包括电脑，手机，平板，都可以通过浏览器看书。如果您需要在外网访问内网的数据，那么您需要购买外网映射服务

2、如何去管理我们的知识体系？我看了一本书，第二天就忘掉了。我们也提供了笔记功能和wiki的功能

3、独学而无友，则孤陋寡闻。所以我考虑在下一个版本中提供教室功能，群组功能，可以做线上活动和读书视频分享会

本应用的目的是促进知识的传播。不过，开发也需要时间和精力的，所以呢，不打算把server作为免费应用。

An Introduction to Lost City's BookSelf
—— An applicatioin born for books

This is a book server for book lovers.

As in the explotion infromations in the world, we should filter these information and build our knowledge system. This is the aim of the product.

Our product should answer the following questions. first, how to manage my ebooks resource. Second, how to build my knowledge system. Third, how to use the system to manage our knowledge.

Inface , I have my ebook files everywhere.In my personal computer, mobile phone, kindle.I also like make annotators and take note.But also do it every where. It's not easy to manage the resource either to form you a better learning system.

So, we provide you a book server to serve your ebooks. in computers even in your mobile.

the server can offer reading books over browsers. support many ebook formats, like .pdf, .djvu, .epub, .mobi, main stream format.Providing annotate, export your annotates.the book server provide api connecting to database(this is advance function).

And what's more? I am considering that providing classroom, group, meeting functons.but it's something to do with the next verson.

how can we manage our knowledge.in fact, we should have interest in our life so that we'll learn it.Build our book system and manage it. By using this, we will raise the interest of gaining knowledges.
